# Đặc tả API

## Tổng quan

- Base URL: `https://chat.hotel-internal.com/api/v1`
- Framework: FastAPI
- Auth: `Authorization: Bearer <jwt_token>` (trừ `/auth/*` và `/widget/*`)
- Response format: JSON
- Streaming: Server-Sent Events (SSE)

---

## 1. Authentication

### POST /auth/google
Đăng nhập qua Google SSO.

```json
// Request
{ "code": "google_auth_code", "redirect_uri": "..." }

// Response
{
  "access_token": "jwt...",
  "refresh_token": "...",
  "expires_in": 900,
  "user": { "id": "uuid", "name": "...", "email": "...", "role": "staff" }
}
```

### POST /auth/refresh
Làm mới access token.

```json
// Request
{ "refresh_token": "..." }

// Response
{ "access_token": "jwt...", "expires_in": 900 }
```

## 2. Chat

### POST /chat/message
Gửi tin nhắn, nhận response dạng SSE stream.

```json
// Request
{
  "session_id": "uuid | null",
  "message": "Chính sách hủy phòng là gì?",
  "channel": "web"
}
```

```
// Response (SSE stream)
data: {"type":"token","content":"Theo chính sách "}
data: {"type":"token","content":"hiện tại..."}
data: {"type":"tool_start","tool":"search_knowledge","input":{"query":"..."}}
data: {"type":"tool_end","tool":"search_knowledge","output":[...]}
data: {"type":"done","session_id":"uuid","tokens_used":420,"model":"claude-haiku-4-5"}
```

### GET /chat/sessions
Danh sách cuộc hội thoại của user hiện tại.

```
GET /chat/sessions?page=1&limit=20&channel=web
```

### GET /chat/sessions/{id}/messages
Lịch sử tin nhắn trong một session.

```
GET /chat/sessions/{id}/messages?before=timestamp&limit=50
```

## 3. Knowledge Base

### POST /knowledge/upload
Upload tài liệu (PDF/DOCX/XLSX). Xử lý bất đồng bộ.

```
Content-Type: multipart/form-data
Fields: file, title, category, access_level, tags[]
```

```json
// Response
{ "doc_id": "uuid", "status": "processing", "message": "Đang xử lý tài liệu..." }
```

Pipeline sau khi upload: `extract text → chunk 512 tokens → embed (Ollama) → lưu pgvector`

### GET /knowledge/search
Tìm kiếm ngữ nghĩa trong knowledge base.

```
GET /knowledge/search?q=chính+sách+hủy+phòng&limit=5&category=policy
```

```json
// Response
{
  "results": [
    { "chunk_text": "...", "doc_title": "...", "category": "policy", "score": 0.92 }
  ]
}
```

### GET /knowledge/docs
Danh sách tài liệu. Query: `?category=policy&page=1&limit=20`

### PUT /knowledge/docs/{id}
Cập nhật metadata tài liệu (title, category, access_level, tags).

### DELETE /knowledge/docs/{id}
Soft delete tài liệu (set `is_active = false`).

## 6. Widget (Embed mode)

### GET /widget.js
File JS bundle để nhúng vào website ngoài. Không cần auth.

### POST /widget/message
Tương tự `/chat/message` nhưng dùng `client_api_key` thay JWT.

```
Headers: X-Widget-Key: CLIENT_API_KEY
```

```json
// Request
{
  "session_id": "uuid | null",
  "message": "Giá phòng loại nào rẻ nhất?",
  "visitor_id": "anonymous_uuid"  // generated by widget JS
}
```

### POST /widget/clients *(admin)*
Tạo client mới, trả về `api_key`.

### POST /widget/clients/{id}/crawl *(admin)*
Trigger crawl website để học nội dung.

```json
{ "url": "https://example.com", "max_pages": 50 }
```

## 9. Error Handling

| HTTP Code | Error Code | Mô tả |
|---|---|---|
| 400 | VALIDATION_ERROR | Request body không hợp lệ |
| 401 | UNAUTHORIZED | Token hết hạn hoặc không hợp lệ |
| 403 | FORBIDDEN | Không có quyền truy cập |
| 404 | NOT_FOUND | Resource không tồn tại |
| 429 | RATE_LIMITED | Vượt quá giới hạn request |
| 500 | INTERNAL_ERROR | Lỗi server |
| 503 | LLM_UNAVAILABLE | Claude API không phản hồi |

```json
// Error response format
{ "error": { "code": "FORBIDDEN", "message": "Bạn không có quyền xem tài liệu này." } }
```

## 10. Rate Limiting

| Endpoint group | Limit | Window |
|---|---|---|
| /chat/message | 30 requests | 1 phút / user |
| /knowledge/search | 60 requests | 1 phút / user |
| /knowledge/upload | 10 requests | 1 giờ / user |
| /widget/message | 60 requests | 1 phút / visitor |
| /webhooks/* | 500 requests | 1 phút / IP |
